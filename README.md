# NPB-align
順位スレのタイトルジェネレータ。<br>
下記URLから、使用できます。<br>
https://onj-ananymouse.github.io/NPB-align/index.html

## 概要
- Claude code でやっつけ作成したWebアプリです。
- 順位をドラッグで入れ替えて、ゲーム差を入力すると自動でスレタイを作成します。
- 「5割の壁」を表示できます。交流戦の勝ち越し数を入力して、補正してください。
- 今日のゲームの勝敗から、ゲーム差とスレタイを更新できます。

## その他、詳細仕様
- MITライセンス
- スマートフォン対応。Sortable.js を使用。
- ローカル動作。サーバーレス。
- キャッシュ対応。前回の入力データをローカルキャッシュに保存します。
  - 「保存します」の表示は、キャッシュへの保存を示すものです。
