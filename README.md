# NPB-align
順位スレのタイトルジェネレータ。
下記URLから、使用できます。
https://onj-ananymouse.github.io/NPB-align/index.html

## 概要
- Claude code でやっつけ作成したWebアプリです。
- 順位をドラッグで入れ替えて、ゲーム差を入力すると自動でスレタイを作成します。
- ローカル + ローカルキャッシュで動きます。
- キャッシュ対応。前回の入力データをローカルキャッシュに保存します。

## 制限事項
- Webからの取得は未対応。
- 5割表示は非対応。気が向いたら作成します。
